ThemeUI
=======

Introduction
------------

Simple theme selector app built using Nui and third party themes.

Default keybindings
-------------------

| Key          | Action              |
|--------------|---------------------|
| `<leader>th` | Open theme selector |
| `<leader>ts` | Toggle theme        |
| `<leader>bg` | Toggle theme mode   |
